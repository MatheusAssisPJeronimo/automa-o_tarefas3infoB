Introdução Sobre HTML

A HTML é uma linguagem de marcação de HiperTexto, ela informa ao navegador que um texto deve ser apresentado de forma diferente, como por exemplo

    list elements
  1. Paragrafo
  2. Tabela
  3. Imagem
  4. Vídeo
  5. Links


Para fazer as marcações usamos Tags(etiquetas) que indicam o inicio e o fim de uma marcação (elemento).

Principais Tags

Tag Objetivo Exemplo
p Cria um Parágrafo

Parágrafo 1

br Cria uma Quebra de linha palavra1
palavra2
h1 Cria um título

Titulo da Página

a Cria um link Google
input Cria um campo de formulário
button Cria um botão
table Cria uma tabela
tr Cria uma linha na tabela
td Cria uma célula na tabela
video Cria um video na pagina <video src="endereço">
img Cria uma imagem <img src="endereço">
